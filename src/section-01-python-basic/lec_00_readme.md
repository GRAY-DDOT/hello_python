# 섹션 1 파이썬 기본 프로그래밍

## 학습 목표

- 파이썬의 기본 문법과 객체 지향 기초, 표준 라이브러리 기초 학습
    - 기본 문법 :
        - 변수, 타입, 연산자, 분기 및 반복
        - 컨테이너 타입, 함수, I/O 기초
    - 객체 지향 기초
        - 클래스
        - 객체 지향의 특성과 기본 원칙
    - 응용 프로그래밍을 위한 추가 학습
        - 모듈, 예외처리
        - 파일, 입출력
